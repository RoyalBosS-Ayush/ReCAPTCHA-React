# Using Google ReCAPTCHA in React

## Procedure

1. **Clone this repo**
    ```sh
    $ git clone https://github.com/RoyalBosS-Ayush/ReCAPTCHA-React
    ```

2. **Get inside the folder**
    ```sh
    $ cd ReCAPTCHA-React/
    ```
3. **Put your SITE_KEY in `Captcha.jsx`**
4. **Run Commands**
    ```sh
    $ npm install
    ```
    ```sh
    $ npm start
    ```
